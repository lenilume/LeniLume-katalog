# LeniLume katalog - Flipbook web project

Ovo je jednostavan statički flipbook projekt generiran iz PDF kataloga. Možeš ga podići na GitHub Pages ili bilo koji static hosting.

Struktura:
- index.html — glavna stranica flipbooka
- style.css — jednostavan CSS za prikaz
- script.js — osnovna logika page switching
- assets/ — slike svake stranice (page_001.png, page_002.png, ...)

Kako objaviti na GitHub Pages:
1. Inicijaliziraj repozitorij: `git init`
2. Dodaj sve datoteke, commit i push na GitHub.
3. U GitHub repo settings, uključi GitHub Pages (branch: main ili gh-pages).

Ako PDF nije renderiran u slike u ovom okruženju, možeš jednostavno uploadati originalni PDF u repozitorij i zamijeniti element u index.html (trenutno se koristi iframe kao fallback).
